## Visión General

**ARK OS Lite** es un sistema operativo ligero basado en Windows 11, diseñado específicamente para dar nueva vida a hardware antiguo o servir como un entorno eficiente para máquinas virtuales.

### Características Clave

- **No Requiere TPM**: Omite el requisito de TPM 2.0 de Windows 11 estándar.
- **Bajo Uso de RAM**: Completamente funcional con tan solo **1 GB de RAM**.
- **Versátil**: Ideal tanto para máquinas virtuales como para hardware legado.
- **Funciona Offline**: Funciona perfectamente con o sin conexión a internet.

## Guía de Instalación

1. Descarga la ISO desde el enlace de arriba.
2. Crea una unidad USB de arranque usando herramientas como Rufus.
3. Arranca tu PC desde la unidad USB.
4. Sigue las instrucciones en pantalla. No se realizará la comprobación de TPM.
